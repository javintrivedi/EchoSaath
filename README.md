<section>
    <h2>📌 Overview</h2>
    <p>
        EchoSaath is a Swift-based iOS application focused on enhancing women's safety.
        It provides instant emergency features, real-time location sharing, and quick alert
        mechanisms to ensure help is always within reach.
    </p>
</section>

<section>
    <h2>🚀 Key Features</h2>
    <ul>
        <li>🚨 <strong>Emergency SOS</strong> – Instantly alert trusted contacts</li>
        <li>📍 <strong>Live Location Sharing</strong> – Share real-time location during emergencies</li>
        <li>📞 <strong>Quick Call Feature</strong> – Call emergency contacts with one tap</li>
        <li>🔊 <strong>Audio/Voice Trigger</strong> – Activate alerts via voice command (if implemented)</li>
        <li>🔐 <strong>Secure Access</strong> – Privacy-focused user data handling</li>
    </ul>
</section>

<section>
    <h2>🛠️ Tech Stack</h2>
    <ul>
        <li><strong>Language:</strong> Swift</li>
        <li><strong>Framework:</strong> SwiftUI / UIKit</li>
        <li><strong>IDE:</strong> Xcode</li>
        <li><strong>APIs:</strong> CoreLocation, Contacts, AVFoundation</li>

</section>
